# 📊 E-commerce Sales Analysis (Python)

## 🚀 Опис проєкту
Проєкт з аналізу e-commerce продажів, виконаний у Python.

Мета — очистити дані, провести дослідницький аналіз (EDA) та отримати бізнес-інсайти щодо продажів, клієнтів і географії.

---

## 📓 Notebook (Google Colab)

Повний аналіз проєкту доступний у Google Colab:

👉 [Відкрити ноутбук](./notebook/Exploratory_Data_Analysis.ipynb)

## 📂 Дані
Проєкт базується на наступних файлах:

- 📊 [events.csv — продажі](./data/events.csv)  
- 📦 [products.csv — категорії товарів](./data/products.csv)  
- 🌍 [countries.csv — географія](./data/countries.csv)
---

## 🧹 Етапи роботи
- Завантаження та об’єднання даних  
- Очищення (пропуски, дублі, типи даних, аномалії)  
- Дослідницький аналіз (EDA)  
- Побудова візуалізацій  

---

## 📊 Аналіз
- Загальні бізнес-метрики (продажі, прибуток, країни)  
- Аналіз категорій товарів  
- Географічний аналіз (країни, регіони)  
- Канали продажів (online / offline)  
- Аналіз динаміки продажів у часі  
- Сезонність (дні тижня)  
- Аналіз часу доставки  

---
